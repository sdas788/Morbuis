## Outcome

Opening a test case surfaces every connected artifact — bugs, runs, Maestro YAML, device coverage, changelog — without leaving the detail panel. All data exists already (E-003, E-004, E-005); this is a UI aggregation epic, not new storage.

## Success Metric

| Metric | Baseline | Target |
|--------|----------|--------|
| Clicks to reach run history from a test case | 3+ (navigate away) | 0 (inline in detail) |
| Visibility of linked bugs from test detail | None | All bugs with any link to this test visible |
