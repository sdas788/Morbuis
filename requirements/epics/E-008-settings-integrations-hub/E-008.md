## Outcome

All team members will have a single settings page to connect devices, configure integrations (Jira, Linear, GitHub, Slack), manage API keys, and monitor the health of all connected services — eliminating the need to edit config files or environment variables directly.

## Success Metric

| Metric | Baseline | Target |
|--------|----------|--------|
| Integrations configurable from UI (vs. config file) | 0 | Jira, BrowserStack, Slack, GitHub all configurable from settings |
| Time to connect a new integration for a non-developer | Manual config edit | Under 5 minutes from Settings page |
