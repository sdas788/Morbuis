## Outcome

A user can create a new Morbius project and import an Excel test plan in the dashboard — no CLI required. Today `morbius import` works from the terminal but the UI has no upload path. This closes the onboarding gap for every new RF engagement.

## Success Metric

| Metric | Baseline | Target |
|--------|----------|--------|
| New projects onboarded without CLI | 0 | 100% of new projects onboard via UI |
| Time from "Create Project" click to seeing populated kanban | N/A (CLI-only) | <60 seconds |

## Primary Persona

The QA Lead (especially at start of a new RF client engagement)
