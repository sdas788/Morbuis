## Outcome

Development teams will be able to run Appium and Detox test suites alongside Maestro flows from the same Morbius dashboard — with a unified results view — eliminating the need to maintain separate tooling for teams that use different test frameworks.

## Success Metric

| Metric | Baseline | Target |
|--------|----------|--------|
| Frameworks supported | 1 (Maestro) | 3 (Maestro + Appium + Detox) |
| Existing Appium suites importable without code changes | 0% | Import and run without modification for standard suites |

## Primary Persona

The Developer

## Dependencies

- E-001 (Device Cloud + BrowserStack) — cloud execution must be available before multi-framework makes sense at scale
- E-002 (Settings + Integrations) — framework adapters configured from Settings page
