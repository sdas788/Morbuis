## Outcome

Any PMAgent project's QA plan can be transferred to Morbius as a real, runnable Morbius project with test cases, in one user action. Each Morbius test case carries backreferences to the source PMAgent story + AC so traceability is preserved. Morbius exposes a results-readback endpoint so PMAgent's future "Reflect" button can render run state next to each AC on the spec side. **Editing PMAgent is out of scope** — we expose a clean API contract; PMAgent wires its Transfer/Reflect buttons later.

The walkthrough that proves the epic: author a QA plan inside PMAgent for the `morbius` project itself (using `pm qa` and/or hand-authoring), click Transfer (or call the API directly today), and a "Morbius" project appears in the dashboard with all test cases, each linked back to its source story.
