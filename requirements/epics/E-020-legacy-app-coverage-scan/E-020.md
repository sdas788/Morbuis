## Outcome

When an old/legacy app enters an engagement with no existing automation plan, Morbius can scan it: upload an Excel of (possibly stale) test cases + point at an app binary → AppMap agent traverses the app → cross-references against uploaded tests → produces a Coverage Gap Report showing what's covered, what's orphaned, and what's high-risk uncovered.

Value proposition: a "diagnostic mode" for picking up legacy client apps where no one knows the current test coverage state.
