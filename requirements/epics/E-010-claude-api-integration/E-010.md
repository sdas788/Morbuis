## Status

**Parked — superseded.** Self-healing scope moves to E-017 (Self-Healing Selectors, Core Four #1). Claude API deep-integration is gated by E-022 (Maestro-as-SDK-Agent, decision-only — do not build). See [direction-2026-04.md](../../wiki/direction-2026-04.md). Do not break into stories.

## Outcome

Any team member — not just developers with Claude Code CLI installed — will be able to use the Morbius AI agent to explore an app, generate all test flows automatically, and have failing tests self-heal when UI selectors change, reducing test maintenance from hours to minutes.

## Success Metric

| Metric | Baseline | Target |
|--------|----------|--------|
| Agent accessible to non-developers (no CLI required) | 0% of team | 100% via Claude API in Settings |
| Selector-change failures auto-fixed without human intervention | 0% | 70% auto-resolved |
| Time to generate flows for a new app screen | Manual YAML authoring (hours) | Under 15 minutes via agent exploration |
