## Steps
Manage Product Locations

## Expected Result
User can view the locations associated with their product. 
User can tap on the location to make changes to the locations associated with their product. 
Validate that after removing all locations and saving changes, user can see and use a CTA to add location to the product. 
Validate that user can add one location to their product and save and see that location added to the product. 
Validate that user can add multiple locations to their product and save and see that location added to the product. 
Validate that user can add a new location and associate it with their product and see that location added to the product. 

Validate that all changes to product locations are displayed on the inventory count screens - group by category and location.
