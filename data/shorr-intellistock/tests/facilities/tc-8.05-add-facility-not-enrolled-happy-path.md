## Expected Result
User can add a new facility to the favorites. 
If facility is not enrolled in the VMI program, it will display greyed out in the search results.
User must confirm that they went to enroll the facility. 
On add, user is prompted to set the facilitily settings. Take note of changes for later validation. 
On save of facility settings, user can add facility locations. 
Add facility locations. Take note of changes later for validation.
