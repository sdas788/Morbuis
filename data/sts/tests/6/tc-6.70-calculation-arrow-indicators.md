## Steps
Calculation Arrow Indicators

## Expected Result
- If a question being answered causes a calculation go up, a red up arrow is displayed next to the calculation indicating the percentage increased 
- If a question being answered causes a calculation go down, a blue down arrow is displayed next to the calculation indicating the percentage decreased 
- If a percentage does not change after an answer is entered, then no arrow is displayed to indicate no change
