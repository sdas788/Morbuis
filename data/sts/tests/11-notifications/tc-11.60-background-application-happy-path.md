## Expected Result
- Send the test message through the backend service layer to your test user.
- The user’s device should ding. A notification banner should be seen from the home screen on iOS, the application icon should be seen left of the status bar on Android.
- Selecting the notification will bring the user to the relevant content within the application.
- If the user selects the application icon to enter the app, the OS notification shade should remove the recently received notification on iOS. This will not happen on some Android OEMs. 
- Ensure in-app notification indicators are showing regardless of the user’s path to enter the application.
