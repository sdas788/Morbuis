## Steps
In-App Notification System

## Expected Result
- Under most circumstances, the user should have a specific place where notifications live within the application
- A notification badge will be used in-app to indicate there is an unread message.
- If the notification is for a post; the user will be shown an indication that a notification is waiting for them. Navigating to the post will clear this visual indication.
- If there’s a mail icon / notification tray; the user can select the icon to see a list of notifications, selecting the specific notification will get rid of the badge
