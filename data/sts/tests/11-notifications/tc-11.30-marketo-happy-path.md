Marketo

## Expected Result
- Within Marketo, you will navigate to Default > Mobile App Testing > App Push Testing > Target All Mobile Users - Prod/Dev > Smart List. This feature will ensure the notifications coming through on your device are following the same flow as a production notification.
- When testing in Prod, ensure you are only sending notifications to your test-user. Utilize a member identification filter to ensure only your test-user is receiving the message.
- You can ensure you are the only test user by selecting Estimate Person Count. The count should equal one.
- Navigate to Schedule and select Run Once. This will send the notification using the filters you’ve set in your Smart List.
- To send another message, navigate to Smart List and then back to Schedule, repeat the previous step.
