## Steps
View Hub Detail – Happy Path (From Dashboard Widget)

## Expected Result
Precondition: user is logged in and has a configured hub with at least one sensor. Open app and land on Dashboard/Hub Home. Tap a hub data widget/tile. User is routed to the Hub Detail page for that hub. The page shows current hub health/status (e.g., connection/online state and any hub-level data points shown in the product). Data displayed matches the selected hub (correct name/ID) and is not blank.
