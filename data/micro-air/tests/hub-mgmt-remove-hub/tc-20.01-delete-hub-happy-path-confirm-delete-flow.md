## Steps
Delete Hub – Happy Path (Confirm Delete)

## Expected Result
Precondition: user is logged in and has at least one configured hub visible on Dashboard/Hub Home. Open app → Dashboard. Open hub kabob menu (⋮) → select Delete. Delete Hub popover appears. User confirms deletion. System deletes the hub from the account/app. User returns to Dashboard/Hub Home and the hub is removed and no longer visible/selectable. Any sensors associated with that hub are removed from dashboard views for that hub.
