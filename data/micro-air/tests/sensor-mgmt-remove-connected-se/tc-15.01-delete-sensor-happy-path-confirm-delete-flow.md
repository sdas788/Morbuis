## Steps
Delete Sensor – Happy Path (Confirm Delete)

## Expected Result
Precondition: user is logged in, hub is configured, and at least one sensor is connected and visible on Dashboard. Open app → Dashboard/Hub Home. Tap a sensor data widget to open Sensor Detail. Open kabob menu (⋮) → select Delete. Confirm Delete overlay appears. User confirms deletion. App shows delete success overlay. User returns to Dashboard/Hub Home and the sensor (tile/widget/data) is removed and no longer selectable.
