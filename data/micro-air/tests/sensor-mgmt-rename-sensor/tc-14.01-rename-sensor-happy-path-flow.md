Rename Sensor – Happy Path

## Expected Result
Precondition: user is logged in, hub is configured, and at least one sensor is connected and visible on Dashboard. Open app → Dashboard/Hub Home. Tap a sensor data widget. App navigates to Sensor Detail page for the selected sensor. Open kabob menu (⋮) → select Rename Sensor. Configure Sensor overlay appears. Update Friendly Name with a valid value and submit/save. System saves the new name to the cloud. User returns to Sensor Detail page and the new friendly name is displayed.
