## Steps
Factory Reset Hub – Happy Path (Confirm Reset)

## Expected Result
Precondition: user is logged in and has a configured hub visible on Dashboard/Hub Home. Open app → Dashboard. Open hub kabob menu (⋮) → select Factory Reset. Factory Reset overlay appears with clear warning text. User confirms reset. System performs factory reset. User sees factory reset success popover. User is returned to Dashboard/Hub Home and the hub is now in a state requiring setup again (e.g., “Connect a Hub” / “Pair hub” callout). User can start the re-pair flow.
