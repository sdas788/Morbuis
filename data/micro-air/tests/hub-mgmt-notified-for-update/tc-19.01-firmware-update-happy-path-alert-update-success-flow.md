## Steps
Firmware Update – Happy Path (Alert → Update → Success)

## Expected Result
Precondition: user is logged in and hub is configured; backend flags hub firmware as out of date. Open app → Dashboard/Hub Home. Dashboard shows a clear alert/indicator that the hub needs a firmware upgrade. User opens hub kabob menu (⋮) → selects Firmware Update. “Firmware update required” overlay appears. User accepts and initiates update (and accepts any required T&C gate if implemented). App enters a waiting/in-progress state until hub returns online. User sees installation success popover displaying the new firmware version #. User returns to Dashboard and hub shows online/connected and no longer shows the update-required alert.
