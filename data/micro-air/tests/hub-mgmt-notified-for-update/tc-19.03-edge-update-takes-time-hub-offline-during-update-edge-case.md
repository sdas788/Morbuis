## Steps
Edge – Update Takes Time / Hub Offline During Update

## Expected Result
Initiate update. Hub temporarily goes offline (expected). App shows a clear progress/waiting state and does not hang indefinitely. If there is a timeout threshold, app exits to a controlled state with retry guidance. When hub comes back, success popover appears and firmware version is updated.
